O comando git add adiciona uma alteração no diretório de trabalho à área de preparação . Ele informa ao Git que você deseja incluir atualizações em um arquivo específico no próximo commit. No entanto, git add não afeta realmente o repositório de maneira significativa - as alterações não são realmente registradas até que você execute git commit .
